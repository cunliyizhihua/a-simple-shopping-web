
----操作 命令行 ----------------------------------------------------------start--

1. 全局安装webpack
	npm install -g webpack

2. 安装依赖
	npm install

3. 预览效果
	windows: npm run dev_win
	mac    : npm run dev_mac

4. 浏览器输入
	localhost:8001/dist/view/index.html
----------------------------------------------------------------------------end--
